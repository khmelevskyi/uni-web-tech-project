Звіти з лабораторних робіт

З дисципліни "Інтернет-технології та проєктування WEB-застосувань"

Студента групи ІС-12 Хмелевського Євгена
Галузь: Онлайн-послуги.
Тема: Сайт студії йоги.

Функціонал:

  • перегляд опису студії;
  • перегляд контактів та соціальних мереж студії;
  • перегляд занять для онлайн тренувань;
  • запис на практичні заняття;
  • перегляд тренерів студії;
  • перегляд основної інформації про йогу та її історію;

Основні сторінки веб-сайту:

  • головна сторінка з описом студії;
  • сторінка для запису на практичні заняття вживу;
  • сторінка з тренерами студії;
  • сторінка з інформацією про йогу та її історію;
  • контакти студії;
  • сторінка з записами занять для онлайн тренувань на дому;
Тема: СТРУКТУРА HTML-ДОКУМЕНТА. ВИБІР ПРЕДМЕТНОЇ ГАЛУЗІ. РОБОТА З ПОСИЛАННЯМИ, ТАБЛИЦЯМИ, ЗОБРАЖЕННЯМИ, СПИСКАМИ в HTML-ДОКУМЕНТІ.
Мета: придбати практичні навички роботи з HTML-документом, таблицями, зображеннями, посиланнями, списками, формами. Створити шаблон звітного HTML-документом для відображення результатів роботи всіх лабораторних робіт.
Місце розташування: Github repository

HTML-документ містить:

  • Head (тут підключено bootstrap)
  • Навігаційна панель (список)
  • Форму для залишення номера телефону на сайті
  • Секція з каруселю
  • Таблицю з посиланнями на соціальні мережі
